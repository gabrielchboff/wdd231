Engineering Inventors Club Website

This website was created as an educational project for WDD 231 - Web Frontend Development course at Brigham Young University. The site showcases a community platform for engineers and inventors to display their innovative projects and connect with fellow innovators.

Project Overview

The Engineering Inventors Club website demonstrates modern web development practices including responsive design, semantic HTML, CSS3 styling, and JavaScript modules with dynamic content loading and local storage implementation.

Data Sources

Projects Data

All project data was created specifically for this demonstration. The projects.json file contains original descriptions and details of fictional engineering projects across multiple categories including robotics, IoT, renewable energy, medical devices, automation, and software/AI applications.

Members Data

Member data is stored in members.json and includes 10 fictional engineer profiles with diverse specialties. Each member profile contains details such as name, specialty, location, skills, achievements, and contact information created for demonstration purposes.

Technologies & Frameworks Used

  • HTML5 - Semantic markup with proper document structure
  • CSS3 - Responsive design with custom properties (CSS variables) and flexbox/grid layouts
  • JavaScript - Modules, arrow functions, template literals, destructuring, and modern APIs
  • Fetch API - Asynchronous data fetching from JSON files with error handling
  • localStorage API - Client-side data persistence for user preferences and application state
  • Modal Dialogs - Accessible modal implementations for displaying detailed information

Design & Styling

The design follows modern web design principles with:

  • Responsive Layout - Mobile-first design supporting devices from 320px to desktop sizes
  • Color Scheme - Professional blue (#1a5490) primary color with orange (#ff6b35) accent and yellow (#ffd23f) highlights
  • Typography - Clean, readable sans-serif font (Segoe UI) with proper contrast ratios for accessibility
  • Accessibility - ARIA labels, semantic HTML, keyboard navigation support, and color contrast compliance
  • Performance - Lazy loading for images, caching strategies, and optimized file sizes

Development Tools & Practices

  • Code Organization - ES6 modules for separation of concerns and code reusability
  • Error Handling - Try-catch blocks for robust asynchronous operation handling
  • Array Methods - Utilization of map(), filter(), reduce(), and forEach() for efficient data processing
  • Template Literals - String interpolation for dynamic HTML generation
  • Event Handling - Comprehensive DOM event listeners and delegation patterns
  • Local Storage - Client-side state management for user preferences and application data

Features Implemented

  • Responsive Navigation with hamburger menu for mobile devices
  • Dynamic content generation from JSON data sources
  • Project filtering by category and status
  • Member search and filtering functionality
  • Modal dialogs for detailed project and member information
  • Form validation and submission with thank-you page
  • Animated statistics on homepage
  • Featured projects carousel/preview
  • Member profile display with skills and achievements
  • Local storage for user preferences and application state

File Structure

  • HTML Files: index.html (home), projects.html, members.html, thank-you.html, attributions.html
  • CSS Files: styles/main.css (global styles), styles/home.css, styles/projects.css, styles/members.css, styles/thankyou.css
  • JavaScript Files: scripts/main.js (shared utilities), scripts/home.js, scripts/projects.js, scripts/members.js, scripts/thankyou.js
  • Data Files: data/projects.json, data/members.json
  • Images: images/ directory containing site assets

Browser Compatibility

This website is designed to work with all modern browsers including:

  • Chrome/Edge (latest)
  • Firefox (latest)
  • Safari (latest)
  • Mobile browsers (iOS Safari, Chrome Mobile)

Credits & Inspiration

This project was developed independently as a comprehensive demonstration of modern web development techniques learned in WDD 231. While inspired by contemporary engineering and technology websites as also old personal projects, all code, content, and design are original creations for educational purposes.
